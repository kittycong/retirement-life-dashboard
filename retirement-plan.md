## 4. 자금 구조

| 구간 | 역할 | 현재 판단 |
|---|---|---|
| 현재~60세 | DC/IRP와 추가저축으로 은퇴 원금 형성 | 월 58만원 자동이체가 기준선 |
| 60~65세 | 국민연금 전 소득공백 구간 | 기준 생활비 기준 약 9,600만원 규모의 브릿지 자금 필요 |
| 65~90세 | 국민연금 + 사적연금 인출 | 국민연금 월 104만원을 제외한 월 66만원 내외 보완 필요 |

## 5. 실행 칸반

| 단계 | 할 일 | 완료 기준 |
|---|---|---|
| 지금 | IRP/연금저축 자동이체 설정 | 월 58만원 기준으로 시작 |
| 이번 분기 | DC/IRP 상품 수수료와 위험자산 비중 점검 | 장기 성장형 중심, 과도한 중복 상품 정리 |
| 매년 | 연금포털 재조회 후 대시보드 숫자 갱신 | 국민연금 예상액, DC/IRP 잔액 업데이트 |
| 50대 | 60~65세 브릿지 자금 분리 | 변동성 낮은 자산으로 3~5년치 생활비 확보 |
| 60세 직전 | 인출 순서 확정 | 퇴직연금, IRP, 일반자산, 국민연금 연결 |

## 6. 우선순위

1. 기준 목표는 월 생활비 170만원으로 두고 월 58만원 추가저축을 시작한다.
2. 세액공제 한도 내에서는 IRP와 연금저축을 우선 사용한다.
3. 60~65세의 소득공백을 별도 목표로 관리한다.
4. 개인연금이 없으므로, 세액공제 계좌와 ISA를 조합해 국민연금 외 현금흐름을 만든다.
5. 매년 5월 전후로 통합연금포털 조회값을 다시 반영한다.

## 7. 웹앱 사용법

`index.html`을 열면 대시보드가 실행된다. 왼쪽 입력 패널에서 생활비, 수익률, 은퇴 나이, 현재 적립금, 국민연금 예상액, 월 추가저축을 조정하면 KPI, 표, 그래프, Markdown 미리보기가 즉시 다시 계산된다.
